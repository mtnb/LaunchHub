## 1. アプリ全体構造

### タブ構成（4タブ）

```
┌─────────────────────────────────────────────┐
│                                             │
│              コンテンツ領域                    │
│                                             │
├──────────┬──────────┬──────────┬─────────────┤
│  🏠 Home │ 🎹 Edit  │ 📁 Lib  │  ⚙ Settings │
└──────────┴──────────┴──────────┴─────────────┘
```

| タブ | 名称 | 説明 | Phase |
|------|------|------|-------|
| Home | ホーム | ダッシュボード、最近のプロジェクト、デバイス状態 | 1 |
| Edit | エディター | ライトショー作成・編集のメイン画面 | 1 |
| Lib | ライブラリ | プリセット・テンプレートの閲覧と管理 | 1 |
| Settings | 設定 | アプリ設定、デバイス管理、MIDI設定 | 1 |

#### Launchpadカラーパレット対応 (128色)

MIDI仕様のパレット値 (0–127) をアプリ内で忠実に再現する。
完全なマッピングは [launchpad-midi-spec.md §カラーパレット](./launchpad-midi-spec.md#カラーパレット-128色) を参照。

### 3.2 タイポグラフィ

SwiftUI Dynamic Type対応。

| 用途 | スタイル | サイズ (デフォルト) | Weight |
|------|---------|-------------------|--------|
| 画面タイトル | `.largeTitle` | 34pt | Bold |
| セクション見出し | `.title3` | 20pt | Semibold |
| パッドラベル | `.caption2` | 11pt | Regular |
| 本文 | `.body` | 17pt | Regular |
| ボタンラベル | `.headline` | 17pt | Semibold |
| 数値入力/BPM | `.system(.monospaced)` | 24pt | Medium |

### 3.3 スペーシング

| トークン | 値 | 用途 |
|---------|-----|------|
| `spacing-xs` | 4pt | パッド間ギャップ |
| `spacing-sm` | 8pt | アイコンとラベルの間 |
| `spacing-md` | 16pt | セクション内余白 |
| `spacing-lg` | 24pt | セクション間余白 |
| `spacing-xl` | 32pt | 画面端パディング (iPhone) |

#### グリッド配置詳細

MIDI仕様 `note = (row * 10) + col` に完全準拠。
ノート番号体系の詳細は [launchpad-midi-spec.md §LEDインデックス体系](./launchpad-midi-spec.md#ledインデックス体系) を参照。

#### ツールバー操作

| ボタン | 機能 | Phase |
|--------|------|-------|
| ↩ Undo | 直前の操作を取り消し | 1 |
| ↪ Redo | 取り消した操作をやり直し | 1 |
| 🎨 色 | カラーピッカーシートを表示 | 1 |
| 💡 モード | LED発光モード切替 (Static/Flashing/Pulsing) | 1 |
| 🔲 選択 | 矩形選択モード (複数パッド一括編集) | 1 |
| 📋 コピー | 選択中のフレームをコピー | 1 |
| LED ドロップダウン | Static / Flashing / Pulsing 切替 | 1 |
| BPM | テンポ入力 (Flash/Pulse速度に連動) | 1 |
| ▶ 再生 | 現在のショーをプレビュー再生 | 1 |
| ⋯ メニュー | エクスポート, 共有, 設定 | 1 |

#### パッドインタラクション

| ジェスチャー | アクション |
|-------------|----------|
| タップ | 選択中のカラーを適用 |
| 長押し | コンテキストメニュー (色変更/モード変更/コピー) |
| 2本指ピンチ | グリッド拡大・縮小 (iPad) |
| スワイプ (パッド上) | 連続塗り (ペイントモード) |

| 要素 | 最小サイズ | 最大スケール | 制約 |

| セクション見出し | 17pt | 3.0x | 制限なし |
| 本文 | 14pt | 3.0x | 制限なし |
| BPM表示 | 20pt | 2.0x | 入力フィールド幅に制約 |

| 状態 | 通常モード | 高コントラストモード |
|------|----------|-------------------|

### 6.4 その他

- **Reduce Motion:** フレーム遷移アニメーションを無効化、即時切替に変更
- **Bold Text:** すべてのテキストウェイトを1段階上げる
- **Switch Control:** すべてのインタラクティブ要素にフォーカス順序を定義
- **カラーフィルタ対応:** パッド色に加えてパターン（斜線/ドット/格子）で色を区別するオプション [Phase 2]

### 7.4 MIDI仕様との対応

UIの各要素がどのMIDIメッセージに対応するかは [launchpad-midi-spec.md](./launchpad-midi-spec.md) を参照。
主な対応: グリッドパッド → Note 11–88, サイドボタン → Note x9, 上部ボタン → CC 104–111,
LED発光モード → MIDI Ch 1/2/3, カラー → パレット(0–127) / RGB(各0–63)。
